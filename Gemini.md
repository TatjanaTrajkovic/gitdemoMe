# Git Demo Me Project

This is a demostration project using **Java 25**

## Project Overview

- **Build System:** Maven
- **Java Version:** 25
- **Main Class:** `com.exemple.gitdemoMe.Main`

## Features

- Uses modern Java 25 features, including instance main methods and implicitly declaration
- Minimalist project structure for demostration purpose.

## How to Run

To compile and the project, use:

```bash
mvn compile exec:java -D"exec.mainClass=com.exemple.gitdemoMe.Main"
```
